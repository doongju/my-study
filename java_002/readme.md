## 클래스와 객체의 정의

- **클래스**
    
    **정의:** 객체를 생성하기 위한 **설계도** 또는 **틀**입니다.
    
    **특징:** 실체로 존재하지 않으며, 어떤 특징(데이터)과 행동(기능)을 가질지만 정의해 둔 상태입니다.
    

- **객체**
    
    **정의:** 클래스라는 설계도를 바탕으로 **메모리에 실제로 구현된 실체**입니다.
    
    **특징:** 각 객체는 자신만의 고유한 상태(데이터)를 가지며, 클래스에 정의된 행동을 수행할 수 있습니다. 프로그래밍에서는 이를 '인스턴스(Instance)'라고도 부릅니다.
    

## 클래스 구성 요소

### ① 필드 (Field / 속성 / 변수)

- **역할:** 객체의 **상태나 데이터**를 저장하는 변수입니다.

### ② 메서드 (Method / 행동 / 함수)

- **역할:** 객체가 할 수 있는 **행동이나 기능**을 정의한 함수입니다. 필드에 값을 넣거나, 필드의 값을 수정하는 등 객체의 데이터를 움직이는 역할을 합니다.

### ③ 생성자 (Constructor)

- **역할:** 객체가 처음 생성될 때 호출되는 특별한 함수로, **객체의 초기 상태를 설정**합니다.
