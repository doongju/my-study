자바의 특징

플랫폼 독립성 : 소스코드를 한 번만 작성하면 윈도우, 맥 , 리눅스 등 어떤 운영체제에서든 수정 없이 그대로 실행할 수 있다. (JVM)

객체 지향 : 모든 것을 객체 단위로 나누어 개발한다. 유지보수가 쉽고 코드 재사용성이 좋다

자동 메모리 관리 : 가바지 컬렉션이 쓰지 않는 메모리를 알아서 정리해준다

활발한 커뮤니케이션 : 오래되긴도 했고 많은 개발자들이 쓰기 때문에 다양한 라이브러리부터 소통이 잘된다

장점
1. os를 구별하지않고 실행이되며 규칙이 엄격하여 안전성이 있다.
2. 하나의 프로그램 안에서 여러 작업을 동시에 처리할 수 있다.

단점
1. 파이썬에 비해 코드 길이가 길고 가독성이 떨어진다
2. 다른 언어들에 비해 속도가 느리고 메모리 소비가 크다
